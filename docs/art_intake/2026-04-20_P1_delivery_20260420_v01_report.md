## 3) Type A 地表瓦片验收（P1必做，但本次仅达成“占位级”）

> Type A 的目标是“纹理”，而不是“场景块/插画块”。  
> 主要判定点：3×3 拼贴是否自然、是否出现大主体重复、是否出现边框/硬切线、是否出现异常冷色或噪点伪影。

### 3.1 关键问题类型（本包中实际出现）

1) **边缘装饰/大主体**（比如草丛/石块沿边一圈）  
   - 结果：重复感非常强，铺图像“贴花边框”。
2) **板块分割/硬边界**（例如石板路、明显分块）  
   - 结果：更像“路面 tile”或“装饰地表”，不属于基础地表纹理。
3) **出现方形框/边界痕迹**（明显非自然纹理）  
   - 结果：直接判定不合格（会穿帮）。
4) **色彩/语义错位**（grass 看起来像灰石裂纹；rock 像石板路）  
   - 结果：即使无缝，也不利于地图语义一致。

### 3.2 初步判定（按“是否可当占位”分级）

> 注：这里“可当占位”并不等于“Type A 合格”，只是允许先跑起来。

#### sand（相对最好）
- `tile_sand_basic_01_w256_h256.png`：✅ 纹理尺度相对合理，边缘干净，可作为占位候选
- `tile_sand_basic_02/03`：⚠️ 边缘/大颗粒偏多，重复感可能明显；建议仅 intake 存档

#### dirt（存在明显问题）
- `tile_dirt_basic_02_w256_h256.png`：❌ 出现明显“方形框/中心区域边界”，不建议使用
- `tile_dirt_basic_01/03`：⚠️ 裂纹/边缘对比偏强，更像插画块；可存档，不建议进 production

#### grass（语义与尺度问题较大）
- `tile_grass_basic_01_w256_h256.png`：❌ 画面偏灰且语义像石裂纹（不建议当 grass）
- `tile_grass_basic_02/03`：⚠️ 边缘装饰+大主体重复感强（存档，不建议进 production）

#### rock（其中一张是“路面”）
- `tile_rock_basic_02_w256_h256.png`：⚠️ 明显石板路，建议改名为 path/road 类，不归 Type A rock
- `tile_rock_basic_01/03`：⚠️ 纹理可用但风格偏冷/噪点强；存档为候选，不建议作为最终 Type A
