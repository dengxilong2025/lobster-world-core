## 2) 统一规格（必须严格满足）

### 2.1 尺寸与格式
- **256×256**
- PNG（RGB 即可；如果做 alpha，也必须合理，不允许大面积误透明）

### 2.2 命名规则（照抄）

**Type A：**
```
tile_{biome}_basic_{variant}_w256_h256.png
```
biome：`grass | dirt | sand | rock | snow | water_shallow`

**Type C（过渡）：**
```
tile_{from}_to_{to}_transition_{variant}_w256_h256.png
```
例：
- `tile_grass_to_dirt_transition_01_w256_h256.png`
