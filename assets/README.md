# Assets

本目录用于存放项目用到的美术/音频等资源。

## 目录约定

- `assets/tiles/generated_from_library/`
  - 由现有风格探索图自动“裁切 + 伪无缝处理”生成的 **临时瓦片**（用于双保险占位）
  - 质量目标：先可用、可替换；后续会被小文的“真正无缝瓦片”替换
- `assets/third_party/`
  - 第三方可商用资源（必须附带来源与许可证）
  - 当前包含 Kenney（CC0）若干包，用作紧急占位与对比参考

## 重要原则
- 任何“生产资源”都要遵循 `docs/art_assets_spec_stoneage_v0_2_draft.md` 的规范（尤其：**禁止在图片里出现任何文字**）。
